- Open-Meteo Forecast: https://api.open-meteo.com/v1/forecast

- REST Countries: https://restcountries.com/v3.1/name/
- Advice Slip: https://api.adviceslip.com/advice
- Cat Fact: https://catfact.ninja/fact
- Dog Facts: https://dogapi.dog/api/v2/facts
- MealDB (random meal): https://www.themealdb.com/api/json/v1/1/random.php
- Agify (age guess): https://api.agify.io
